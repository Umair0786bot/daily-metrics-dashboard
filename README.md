# Amazon Daily Metrics Dashboard

Live dashboard for the DECOLURE / SLEEPHORIA Amazon **US** account — daily performance
across the full catalog plus the **LTSF (Aged Inventory)** tracker.

## View it

Open **`index.html`** — it is fully self-contained (no internet or install needed).
Live link (GitHub Pages): the repo's Pages URL.

- **Full Catalog** tab: 12 parents · 725 child SKUs — sales, PPC, ACOS/TACOS, CVR, drill-down to any SKU.
- **LTSF · Aged Inventory** tab: 23 aged parents · 517 SKUs — AIS charge exposure, sell-through velocity, days-to-clear.

Date ranges 7 / 14 / 30 / 90 / All · light + dark · sortable tables.

## Updating

This page is regenerated by the local tracker (`Update-Dashboard.bat`) and pushed here
automatically. Do not edit `index.html` by hand.
